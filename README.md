# canvas-benchmarks

https://benchmark.leaferjs.com

Performance Benchmarking for Mainstream Canvas Engines - Currently Included Scenarios:

[PixiJS](https://pixijs.com)、[Konva.js](https://konvajs.org)、[Fabric.js](http://fabricjs.com)、SVG、HTML、[LeaferJS](https://www.leaferjs.com/)

Provides performance test comparisons for practical use cases including element creation, viewport scaling, viewport panning, element dragging, fully dynamic elements, large-scale image zooming, and more.

主流 canvas 引擎的性能基准测试，目前已收录的有：

[PixiJS](https://pixijs.com)、[Konva.js](https://konvajs.org)、[Fabric.js](http://fabricjs.com)、SVG、HTML、[LeaferJS](https://www.leaferjs.com/)

提供创建元素、缩放视图、平移视图、拖拽元素、全动态元素、大图缩放等实用场景的性能测试对比。
